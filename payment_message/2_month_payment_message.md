Изучение SQL.
Для обучения SQL не выделяются отдельные спринты. Его мы изучаем параллельно с Pandas. Почему? SQL обрабатывает таблицы, Pandas тоже. Методы SQL близки методам Pandas, и то, и то - инструменты обработки и анализа данных. Кроме того, изучение SQL - это всегда решение большого количества задач, которое вполне можно выполнить на специальном тренажере.
