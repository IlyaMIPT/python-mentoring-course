# Найти все анаграммы

Даны две строки s и p. Найдите в строке s все начальные индексы подстрок, которые являются анаграммами строки p. Выведите список этих индексов.

Вход: s = "cbaebabacd", p = "abc"
Выход: [0, 6]
Пояснение: Подстрока, начинающаяся с индекса 0 — "cba", с индекса 6 — "bac", обе являются анаграммами "abc".


# Разделить список

Дан список целых чисел nums и целое число pivot. Преобразуйте список так, чтобы сначала шли все элементы меньше pivot, затем равные pivot, и в конце — больше pivot.

Пример:
Вход: nums = [9, 12, 3, 5, 14, 10, 10], pivot = 10
Выход (один из возможных): [9, 3, 5, 10, 10, 12, 14]


# Полный квадрат

Напишите функцию, которая возвращает true, если число является полным квадратом (квадратом целого числа).
Число является полным квадратом, если существует такое целое число n, что n * n равно заданному числу.
Например, число 9 является полным квадратом, так как 3 * 3 = 9.
Число 10 полным квадратом не является, потому что нет целого числа, квадрат которого равен 10.


# Степерь двойки

Напишите функцию, которая возвращает true, если число является степенью двойки.
Число является степенью двойки, если существует такое целое неотрицательное число k, что 2^k равно заданному числу.
Например, числа 1, 2, 4, 8, 16 являются степенями двойки.
Подсказка: используйте цикл с делением на 2.


# Палиндромы в строке

Даны две строки s и p.
Найдите в строке s все начальные индексы подстрок фиксированной длины len(p), которые являются палиндромами.
Выведите список этих индексов в порядке возрастания.

Вход:
s = "abacab"
p = "aba"
Длина подстроки равна 3.
Выход: [0, 4]
Пояснение:

Подстрока s[0:3] = "aba" — палиндром.

Подстрока s[4:7] = "bab" — палиндром (если индекс 4 соответствует "b", то s[4:7]="bab").
